# test_word_cloud

## 概要

テキストを描画するツールです。 
描画ツールは、[word_cloud](https://github.com/amueller/word_cloud) を使っています。 

## 実行方法

```
$ git clone git@github.com:yagiyuki/text_drawing.git
$ cd text_drawing/
$ python run.py 
```

## 依存ライブラリ

* [word_cloud](https://github.com/amueller/word_cloud)
* [numpy](https://github.com/numpy/numpy)
